    Python数据分析基础¶

    发布日期: 2024-03-12
    分类: 数据科学
    标签: Python, 数据分析, Pandas, NumPy, 数据可视化

    引言¶

    在当今数据驱动的世界中，数据分析能力已经成为各行各业的核心竞争力。Python凭借其简洁的语法和丰富的库，已经成为数据分析领域的首选语言之一。本文将介绍Python数据分析的基础知识，包括常用的库、数据处理技巧和可视化方法。

    数据分析工具包¶

    NumPy: 数值计算的基础¶

    NumPy是Python科学计算的基础库，提供了高性能的多维数组对象和处理这些数组的工具。

    import numpy as np

# 创建数组
arr = np.array([1, 2, 3, 4, 5])
print(arr)  # [1 2 3 4 5]

# 数组运算
print(arr * 2)  # [2 4 6 8 10]

# 统计函数
print(np.mean(arr))  # 3.0
print(np.std(arr))   # 1.4142135623730951

    Pandas: 数据处理和分析¶

    Pandas提供了高性能、易用的数据结构和数据分析工具，特别适合处理表格数据。

    import pandas as pd

# 创建DataFrame
data = {
    'Name': ['John', 'Anna', 'Peter', 'Linda'],
    'Age': [28, 24, 35, 32],
    'City': ['New York', 'Paris', 'Berlin', 'London']
}
df = pd.DataFrame(data)
print(df)

    输出: 

        Name  Age      City
0   John   28  New York
1   Anna   24     Paris
2  Peter   35    Berlin
3  Linda   32    London

    Matplotlib: 数据可视化¶

    Matplotlib是Python最流行的绘图库，提供了一整套用于创建静态、动态和交互式可视化的工具。

    import matplotlib.pyplot as plt

# 简单折线图
x = np.linspace(0, 10, 100)
y = np.sin(x)
plt.figure(figsize=(10, 6))
plt.plot(x, y, 'b-', linewidth=2)
plt.title('Sine Wave')
plt.xlabel('x')
plt.ylabel('sin(x)')
plt.grid(True)
plt.show()

    Seaborn: 统计数据可视化¶

    Seaborn是基于Matplotlib的高级可视化库，提供了更美观的默认样式和更高级的统计图表。

    import seaborn as sns

# 设置样式
sns.set(style="whitegrid")

# 加载示例数据集
tips = sns.load_dataset("tips")

# 创建小提琴图
plt.figure(figsize=(10, 6))
sns.violinplot(x="day", y="total_bill", hue="sex", data=tips, split=True)
plt.title('Total Bill Distribution by Day and Gender')
plt.show()

    数据处理实战¶

    数据加载和探索¶

    # 加载CSV文件
df = pd.read_csv('data.csv')

# 查看数据基本信息
print(df.info())
print(df.describe())

# 查看前几行数据
print(df.head())

# 检查缺失值
print(df.isnull().sum())

    数据清洗¶

    # 处理缺失值
df.fillna(df.mean(), inplace=True)  # 用均值填充数值型缺失值
df['category'].fillna('Unknown', inplace=True)  # 用特定值填充分类型缺失值

# 删除重复行
df.drop_duplicates(inplace=True)

# 异常值处理
Q1 = df['value'].quantile(0.25)
Q3 = df['value'].quantile(0.75)
IQR = Q3 - Q1
df = df[~((df['value'] < (Q1 - 1.5 * IQR)) | (df['value'] > (Q3 + 1.5 * IQR)))]

    数据转换¶

    # 类型转换
df['date'] = pd.to_datetime(df['date'])
df['category'] = df['category'].astype('category')

# 创建新特征
df['year'] = df['date'].dt.year
df['month'] = df['date'].dt.month
df['day_of_week'] = df['date'].dt.day_name()

# 标准化数值特征
from sklearn.preprocessing import StandardScaler
scaler = StandardScaler()
df[['value1', 'value2']] = scaler.fit_transform(df[['value1', 'value2']])

    数据聚合和分组¶

    # 分组统计
group_stats = df.groupby('category').agg({
    'value': ['mean', 'std', 'min', 'max', 'count']
})
print(group_stats)

# 透视表
pivot = pd.pivot_table(df, values='value', index='category', 
                       columns='year', aggfunc='mean')
print(pivot)

    数据可视化案例¶

    探索性数据分析¶

    # 分布图
plt.figure(figsize=(12, 6))
plt.subplot(1, 2, 1)
sns.histplot(df['value1'], kde=True)
plt.title('Distribution of Value1')

plt.subplot(1, 2, 2)
sns.boxplot(x='category', y='value1', data=df)
plt.title('Value1 by Category')
plt.tight_layout()
plt.show()

# 相关性分析
plt.figure(figsize=(10, 8))
corr = df.select_dtypes(include=[np.number]).corr()
sns.heatmap(corr, annot=True, cmap='coolwarm', fmt='.2f')
plt.title('Correlation Matrix')
plt.show()

    时间序列可视化¶

    # 按时间聚合数据
time_series = df.groupby(pd.Grouper(key='date', freq='M'))['value'].mean()

# 绘制时间序列图
plt.figure(figsize=(12, 6))
time_series.plot(marker='o')
plt.title('Monthly Average Value')
plt.xlabel('Date')
plt.ylabel('Average Value')
plt.grid(True)
plt.show()

    实际项目案例：销售数据分析¶

    假设我们有一个电商平台的销售数据，包含订单日期、产品类别、销售额等信息。我们想要分析销售趋势和产品表现。

    # 加载数据
sales_df = pd.read_csv('sales_data.csv')
sales_df['order_date'] = pd.to_datetime(sales_df['order_date'])

# 按月份和产品类别分析销售额
monthly_sales = sales_df.groupby([pd.Grouper(key='order_date', freq='M'), 'product_category'])['sales_amount'].sum().reset_index()

# 可视化月度销售趋势
plt.figure(figsize=(14, 7))
for category, data in monthly_sales.groupby('product_category'):
    plt.plot(data['order_date'], data['sales_amount'], marker='o', linewidth=2, label=category)

plt.title('Monthly Sales by Product Category')
plt.xlabel('Month')
plt.ylabel('Sales Amount')
plt.legend()
plt.grid(True)
plt.xticks(rotation=45)
plt.tight_layout()
plt.show()

# 产品类别销售占比
plt.figure(figsize=(10, 7))
category_sales = sales_df.groupby('product_category')['sales_amount'].sum()
plt.pie(category_sales, labels=category_sales.index, autopct='%1.1f%%', startangle=90, shadow=True)
plt.title('Sales Distribution by Product Category')
plt.axis('equal')
plt.show()

    进阶技巧¶

    使用apply和lambda函数¶

    # 对DataFrame应用自定义函数
df['sales_category'] = df['sales_amount'].apply(
    lambda x: 'High' if x > 1000 else ('Medium' if x > 500 else 'Low')
)

    使用merge合并数据¶

    # 合并两个DataFrame
customers = pd.DataFrame({
    'customer_id': [1, 2, 3, 4],
    'name': ['John', 'Anna', 'Peter', 'Linda']
})

orders = pd.DataFrame({
    'order_id': [101, 102, 103],
    'customer_id': [1, 3, 3],
    'amount': [150, 200, 300]
})

# 内连接
customer_orders = pd.merge(customers, orders, on='customer_id')
print(customer_orders)

    使用多进程加速数据处理¶

    from multiprocessing import Pool
import pandas as pd

def process_chunk(chunk):
    # 对数据块进行处理
    result = chunk['value'] * 2
    return result

# 将大型DataFrame分割成块并并行处理
def parallel_process(df, func, n_cores=4):
    df_split = np.array_split(df, n_cores)
    pool = Pool(n_cores)
    df['result'] = pd.concat(pool.map(func, df_split))
    pool.close()
    pool.join()
    return df
